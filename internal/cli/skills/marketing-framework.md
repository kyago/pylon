# Marketing Campaign Framework

## Campaign Strategy
1. **Objective** — Define measurable goal (awareness, leads, conversion)
2. **Audience** — Segment by demographics, behavior, and intent
3. **Positioning** — Differentiate from competitors with unique value
4. **Channels** — Select channels matching audience behavior
5. **Budget** — Allocate based on channel ROI expectations

## Funnel Analysis
| Stage | Metric | Optimization |
|-------|--------|-------------|
| Awareness | Impressions, Reach | Creative testing, targeting |
| Interest | CTR, Engagement | Messaging, ad format |
| Consideration | Site visits, Time on page | Landing page, content |
| Conversion | CVR, CPA | CTA, offer, UX |
| Retention | LTV, Repeat rate | Email, loyalty, support |

## KPI Framework
- **North Star Metric**: One metric that best captures value delivery
- **Leading indicators**: Predictive metrics (traffic, engagement)
- **Lagging indicators**: Outcome metrics (revenue, retention)
- Set targets with baseline, target, and stretch goals

## A/B Testing Protocol
1. Form hypothesis: "Changing X will improve Y by Z%"
2. Define control and variant
3. Calculate required sample size for statistical significance
4. Run test for minimum duration (avoid peeking)
5. Analyze results with confidence interval
6. Document learnings regardless of outcome
