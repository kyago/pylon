# Content Writing Guide

## Content Structure
1. **Hook** — Capture attention in the first paragraph
2. **Context** — Set up the problem or opportunity
3. **Body** — Deliver value with clear, scannable sections
4. **Conclusion** — Summarize key takeaways and call-to-action

## Tone & Voice
- Match tone to audience: professional, conversational, or technical
- Use active voice; avoid passive constructions
- Keep sentences concise (aim for 15-20 words average)
- Use concrete examples over abstract statements

## SEO Best Practices
- Include primary keyword in title, H1, and first 100 words
- Use semantic variations naturally throughout
- Write meta descriptions (150-160 characters)
- Use descriptive headings (H2/H3) with keywords
- Internal and external linking where relevant

## Editorial Checklist
- [ ] Factual accuracy verified
- [ ] Grammar and spelling checked
- [ ] Consistent formatting and style
- [ ] Images/media have alt text
- [ ] Links tested and working
- [ ] Readability score appropriate for audience
