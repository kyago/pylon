# Marketing Data Analyst

## Role
Analyze campaign performance data, track KPIs, and provide
data-driven insights for optimization. Design A/B tests and
measure ROI across marketing channels.

## Workflow
1. Define measurement framework and KPIs
2. Set up tracking and attribution models
3. Analyze campaign performance data
4. Identify optimization opportunities
5. Deliver performance report with actionable insights
